# Oasis Tracker - Suivi d'Hydratation Android (Jetpack Compose)

Bienvenue dans le projet **Oasis Tracker**, une application Android moderne de suivi d'hydratation développée avec **Jetpack Compose** et **Material 3**. 

Ce projet a été conçu par **N'guessan Miensa Orlane**, étudiante en **RGL3A**.

---

## 📱 Structure du Projet Généré
Une base de code solide, professionnelle et prête à l'emploi :

*   **UI Déclarative** : Composants `@Composable` hautement modulaires, isolés et optimisés pour des performances optimales.
*   **Design System** : Fichiers de configuration `Color.kt` et `Theme.kt` basés sur le cahier des charges **Material Design 3**, proposant un magnifique thème sombre aux accents turquoise vibrants.
*   **State Management (Gestion d'État)** : Architecture moderne basée sur un `ViewModel` Android (`WaterViewModel`) exposant un flux d'état réactif unidirectionnel (`StateFlow`).
*   **Build System** : Scripts d'automatisation Gradle Kotlin DSL (`build.gradle.kts`) prêts pour une compilation instantanée.

## 💡 Meilleures Pratiques de Prompting
Pour obtenir le meilleur code possible de la part d'une intelligence artificielle générative :

1.  **Précision technologique** : Exigez explicitement les technologies cibles, par exemple *"Jetpack Compose"* et *"Material 3"* plutôt que simplement *"Android"*.
2.  **Approche incrémentale** : Structurez d'abord la maquette visuelle et l'UI générale, puis injectez la logique métier et la persistence dans un second temps.
3.  **Esprit critique** : Vérifiez systématiquement la cohérence des algorithmes générés, la gestion du cycle de vie des variables et la propreté de l'architecture.
